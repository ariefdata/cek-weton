# Cek Weton

Aplikasi web sederhana untuk menghitung weton dari tanggal lahir.

Fitur saat ini:
- Hitung hari, pasaran, wuku, dan neptu
- Menampilkan interpretasi sederhana dari data JSON
- Menampilkan medan energi hari ini secara otomatis
- Tema tampilan: Klasik, Royal, Malam

## Menjalankan

Gunakan live server atau buka `index.html` langsung di browser.

## Struktur

- `index.html` UI utama
- `script.js` logika perhitungan
- `css/style.css` styling
- `data/interpretasi.json` data interpretasi

## Catatan

Interpretasi saat ini bersifat umum dan bisa diganti sesuai referensi yang kamu gunakan.
